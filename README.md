# 🤖 Бот технической поддержки HDL для Битрикс24

Бот для поиска документации и технической информации по продуктам HDL, Buspro, KNX, EasyCool и другим брендам.

## 📋 Функциональность

- 🔍 Поиск по индексу из 10,000+ файлов
- 📁 Перенаправление на папки Яндекс.Диска
- 🔄 Синонимы и нормализация запросов
- 🎯 Контекстный поиск с уточнениями
- 💬 Интерактивные кнопки в чате
- 👨‍💻 Перевод на специалиста
- ⭐ Оценка полезности ответов

# 🤖 Бот технической поддержки HDL для Битрикс24

Бот для поиска документации и технической информации по продуктам HDL, Buspro, KNX, EasyCool и другим брендам.

## 📋 Функциональность

- 🔍 Поиск по индексу из 10,000+ файлов
- 📁 Перенаправление на папки Яндекс.Диска
- 🔄 Синонимы и нормализация запросов
- 🎯 Контекстный поиск с уточнениями
- 💬 Текстовые команды для управления
- 👨‍💻 Перевод на специалиста (ID: 39)
- ⭐ Оценка полезности ответов

## 📝 Команды бота

| Команда | Описание |
|---------|----------|
| `/refine` | Уточнить запрос |
| `/transfer` | Связаться со специалистом |
| `/helpful` | Отметить ответ полезным |
| `/categories` | Показать категории |
| `/more` | Показать больше результатов |
| `/test` | Проверить работу бота |

## 🔍 Примеры запросов

- `кабель` - документация по кабелям
- `замок` - дверные замки iOT Systems
- `easycool техничка` - документация EasyCool
- `карниз buspro` - карнизы Buspro
- `интеграция с алисой` - голосовые помощники
- `urri` - документация URRI
